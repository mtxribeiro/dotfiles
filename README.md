## Aviso

Não contém drivers de GPU.
O script pode sobrescrever seus arquivos de configuração.
É altamente recomendado fazer um backup.

## Dependências

sudo pacman -S git fish
chsh -s /bin/fish
fish

## Instalação

git clone https://github.com/mtxribeiro/dotfiles.git
cd ~/dotfiles
./install.sh
